# Secure Password Manager

A secure full stack password management web application developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The application helps users securely store and manage credentials for different websites and platforms using authentication and encrypted password protection.

---

## Features

* Secure user authentication system
* Password encryption using bcrypt
* Add, update, delete, and manage credentials
* Responsive and user-friendly interface
* Form validation for secure data handling
* MongoDB integration for efficient database management
* CRUD operations implementation
* Sorting and projection using MongoDB queries
